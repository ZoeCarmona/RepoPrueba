# Página Prueba

Este proyecto es una página web simple (HTML + CSS) con un diseño limpio tipo “landing” y una frase aleatoria que cambia cada vez que recargas.

## ¿Qué incluye?
- `index.html`: estructura de la página
- `style.css`: estilos básicos y responsivos
- Frases aleatorias con JavaScript (incluido dentro del HTML)
